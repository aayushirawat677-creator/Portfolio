Lumi Body Doubling — presence after the push.

Lumi is great at getting users over the start friction. Then they get bored, lose momentum, and quit halfway. This PRD turns the voice channel we're already on into an on-demand body-doubling companion.

01 · The Problem

We're hanging up too early.

The 5-minute persuasive call works — users start the task. But once they're actually doing it, momentum trails off and they reach for their phone. Lumi is already on the call. We just leave.

From interviews, the pattern was consistent: users get started thanks to Lumi, then lose momentum mid-task and quit. Body doubling — working alongside someone else — is one of the most evidence-backed techniques for sustaining engagement through the hard middle of a task. In our user survey, 75% said they already do it elsewhere — on Focusmate, Flow Club and focus communities — paying for it, scheduling around it, and asking why Lumi can't.

Reminders, habit trackers and rewards only front-load motivation. They don't provide ongoing presence, so motivation trails off after the first few minutes. Lumi already solves start friction — what's missing is the continuity of presence, which is exactly what body doubling provides.

03 · What We're Building

Two modes — and a clean exit.

When a user confirms "I'm starting," Lumi transitions out of intervention and into a body-doubling session, behaving differently based on the task.

// mode 01

Chat mode

Lumi makes light conversation, comments on what the user's doing, talks about its own "work." For chores, cooking, exercise. Goes quiet 2–3 min → "Still with me?"

// mode 02

Deep Focus mode

For studying or focused work. Lumi stays on the call but doesn't speak — plays ambient audio (forest, café, campfire). The user actively ends the session.

// path 03

Leaving the house

Gym, errands, school drop-off: Lumi guides the prep, then cuts the call the moment the user clicks "I'm starting." Body doubling only applies to in-place tasks.

04 · Use Cases

How a session actually feels.

01

Chat — folding laundry

"I'll just be over here drafting some notes. Tell me when you're ready." Light banter as the user folds; gentle re-engagement after silence; ends on "I'm done."

02

Deep Focus — studying

"Putting the headphones on. Forest, café, or campfire?" Voice channel stays connected but silent; ambient audio loops; VAD wakes Gemini Live when the user speaks to end or switch.

03

Mid-session switch

"Lumi, switch to deep focus — I want to think." Lumi stops speaking, ambient audio starts at the last-used track, resumes voice only to close out.

04

Interruptions & exits

Heading to the gym → call cuts on "I'm starting." Taking a phone call → session pauses, resumes on return: "You're back. How'd that go?"

05 · How It Works

VAD is the cost-critical piece.

The whole economic model of Deep Focus mode hinges on waking the expensive model only when needed.

// frontend

Mode picker & session UI

Default inferred from task type (chores → Chat, study → Deep Focus), user can override. Minimal active-session screen — readable when the phone's propped at arm's length. Locally-bundled ambient audio, 3 tracks.

// backend

Gemini Live connection mgmt

Chat mode: connection open the full session. Deep Focus: closes once "I'm starting," reopens only when VAD detects speech. A session state machine tracks mode, elapsed time, and last interaction.

// VAD

Relative-threshold detection

Wake on volume above the rolling ambient average, not an absolute level — so it handles quiet bedrooms and noisy cafés without per-user retuning. Reconnect must land under 2 seconds or the experience breaks.

08 · Risks

What could break it.

!
VAD false-positivesRandom wakes during studying kill Deep Focus. Conservative Phase-1 threshold; telemetry on every wake event.
!
Gemini Live cost overrunIf VAD underperforms, Deep Focus costs as much as Chat. Hard 60-min cap; monitor cost-per-session daily.
!
Doubling doesn't lift completionThe existential risk. If completion isn't better than non-doubled tasks, revert to intervention-only and keep the learnings.
!
Users prefer humansThe 75% already using Focusmate and focus communities may value the human element. Position Lumi as on-demand and complementary, not a replacement.
